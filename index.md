# Bem Vindo!

## 📝 Sobre este site

Uma base de conhecimento e blog para profissionais de Software de engenheiros de software a tech leads, com melhores práticas, guias de otimização de desempenho, técnicas de revisão de código e princípios de liderança técnica.

## 🤓 Sobre mim

Me chamo Lucas, sou um engenheiro de software apaixonado por tecnologia desde criança e achei que seria legal criar esse site para deixar público meus guias e notas, contribuindo com a comunidade de tecnologia.
